CSS Structure Update Guide

This guide explains the new CSS structure for the TrendyTees website. The CSS has been organized into a modular structure for better maintainability and organization.

New CSS File Structure

  • css/
    • main.css (imports all other CSS files)
    • base/
      • variables.css (color variables, spacing, etc.)
      • reset.css (CSS reset and base styles)
      • typography.css (text styles, headings)
      • utilities.css (utility classes)
    • layout/
      • header.css (header styles)
      • footer.css (footer styles)
    • components/
      • hero.css (hero section)
      • categories.css (category cards)
      • products.css (product cards)
      • colors.css (color dot styles)
      • offers.css (special offers section)
      • newsletter.css (newsletter section)
      • filters.css (category page filters)
    • pages/
      • home.css (home page specific styles)
      • category.css (category pages styles)

How to Update Your HTML Files

To use the new CSS structure, update the link tag in your HTML files from:

<link rel="stylesheet" href="css/styles.css">

To:

<link rel="stylesheet" href="css/main.css">

For pages in the "pages" directory, update from:

<link rel="stylesheet" href="../css/styles.css">

To:

<link rel="stylesheet" href="../css/main.css">

Benefits of the New Structure

  • Better organization of styles by purpose
  • Easier to find and update specific components
  • Reduced CSS conflicts
  • More maintainable codebase
  • Faster development for new features
