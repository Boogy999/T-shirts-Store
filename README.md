# TrendyTees E-Commerce Website

A modern e-commerce website for selling t-shirts for men, women, boys, and girls.

## Project Overview

TrendyTees is a responsive e-commerce platform with a focus on usability and modern design. The website features:

- Four main categories: Men, Women, Boys, and Girls
- Modern and colorful design
- Responsive layout for all devices
- Interactive product displays
- Shopping cart functionality
- Newsletter subscription

## Features

- **Home Page**: Showcases featured products from all four categories
- **Category Sections**: Dedicated sections for Men, Women, Boys, and Girls
- **Product Cards**: Display product images, prices, color options, and add-to-cart functionality
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic cart updates
- **Local Storage**: Cart data is saved in the browser's local storage

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Navigate through the website to explore all features
